# 04 · البناء

في هذه المرحلة يتم تحويل الهيكلة المعتمدة إلى موقع ثابت متجاوب وجاهز للاستخدام. ويتم إنتاج جميع المخرجات المؤكدة، بما فيها تحويل التقرير إلى HTML إذا كان ضمن النطاق.

## مخرجات التقارير

في الباقة 2 والباقة 3، دمج التقرير ضمن المشروع يكون مشمول. الصيغة تعتمد على ما تم اعتماده في مرحلة الاستقبال.

<div class="card-grid">

  <article class="info-card">
    <h3>PDF مدمج</h3>
    <p>يتم تضمين التقرير الحالي داخل صفحة مخصصة في الموقع ويمكن تصفحه مباشرة بدون تنزيل.</p>
  </article>

  <article class="info-card">
    <h3>تقرير HTML مستضاف</h3>
    <p>يتم تحويل التقرير إلى صفحة HTML منسقة ومستضافة داخل الموقع وقابلة للقراءة على جميع الأجهزة.</p>
  </article>

  <article class="info-card">
    <h3>تقرير HTML مستقل</h3>
    <p>يتم تسليم التقرير كملف HTML مستقل للتنزيل أو الاستضافة المنفصلة، بدون إدراجه في تنقل الموقع.</p>
  </article>

</div>

يجب تسليم التقرير كامل قبل بداية مرحلة البناء. التسليم الجزئي يسبب تأخير في التنفيذ.

---

## مخرجات هذه المرحلة

- موقع ثابت مكتمل يعكس الهيكلة المعتمدة
- دمج كامل للمحتوى وتنسيقه
- إنتاج جميع المخرجات المتفق عليها
- نسخة جاهزة للمراجعة
