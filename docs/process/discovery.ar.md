# 02 · الاكتشاف

مرحلة الاكتشاف تحدد كل شيء سيتم بناؤه. يتم فيها اعتماد الهدف والجمهور وجاهزية المحتوى والأقسام المطلوبة. بعد اكتمالها، يتم تثبيت النطاق.

---

  <article class="hero-card">

## مخرجات هذه المرحلة

- نطاق مشروع معتمد وتأكيد الباقة  
- قائمة أقسام نهائية  
- جرد للمحتوى  
- اتجاه هيكلي معتمد  

يتم تثبيت النطاق بعد هذه المرحلة. أي تغييرات جديدة بعد الاكتشاف تراجع وتسعر بشكل مستقل قبل التنفيذ.
