# 04 · Build

The approved structure is implemented into a fully functional, responsive static website. All confirmed deliverables, including HTML report conversion if included, are produced during this stage.

## Report Deliverables

For Tier 2 and Tier 3 projects, report integration is included. The format depends on what was confirmed during intake.

<div class="card-grid">

  <article class="info-card">
    <h3>Embedded PDF</h3>
    <p>The existing report is embedded directly into a dedicated page on the site and accessible without leaving the site or downloading a file.</p>
  </article>

  <article class="info-card">
    <h3>HTML Report: Hosted</h3>
    <p>The report is converted into a fully styled HTML page and hosted as part of the site. Readable on any device without a PDF viewer.</p>
  </article>

  <article class="info-card">
    <h3>HTML Report: Standalone</h3>
    <p>The report is converted into a self-contained HTML file and delivered for download or independent hosting. Not integrated into the site navigation.</p>
  </article>

</div>

The original report document must be submitted in full before the build stage begins. Partial submissions will delay delivery.
