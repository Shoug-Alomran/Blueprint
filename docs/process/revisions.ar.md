# 05 · المراجعة والتحسينات

بعد اكتمال البناء، تتم مراجعة الموقع مقابل الأهداف والهيكلة المعتمدة. يتم جمع الملاحظات وتقييمها وتنفيذها ضمن عدد الجولات المشمولة.

المراجعات للتحسين، وليست لإعادة التصميم.

## جولات المراجعة حسب الباقة

| الباقة | عدد الجولات المشمولة |
| --- | --- |
| الباقة 1 – حضور شخصي | جولتان |
| الباقة 2 – توثيق مشروع | جولتان |
| الباقة 3 – توثيق بحثي | 3 جولات |

يمكن طلب جولات إضافية بعد الجولات المشمولة بسعر 100 ريال لكل جولة، ويتم التأكيد قبل التنفيذ.

---

## حماية النطاق

المراجعات تقتصر على تحسينات داخل الهيكلة والنطاق المعتمدين. الحالات التالية تحتاج تسعير مستقل قبل التنفيذ:

- إضافة أقسام أو صفحات جديدة غير مشمولة في النطاق الأصلي  
- إعادة هيكلة كاملة أو تعديل جذري في التنقل  
- مخرجات جديدة لم يتم تأكيدها في مرحلة الاستقبال  

---

## مخرجات هذه المرحلة

نسخة مصقولة وجاهزة للاعتماد النهائي، متوافقة مع الأهداف الأصلية والهيكلة المعتمدة وملاحظات العميل.
