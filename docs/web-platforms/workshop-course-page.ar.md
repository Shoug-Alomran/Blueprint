# دراسة حالة — صفحة ورشة أو دورة

## السياق

المطلوب صفحة تنظّم البرنامج والمصادر بشكل واضح للمشاركين.

## التحدي

- تشتت الروابط والمعلومات
- صعوبة الوصول السريع للمصادر

## الحل

- ترتيب المحتوى حسب الرحلة: قبل الورشة، أثناءها، وبعدها
- إبراز الجدول الزمني والروابط المهمة
- تصميم خفيف وسريع

## النتيجة

- تجربة أوضح للمشارك
- تقليل الأسئلة المتكررة
- وصول أسرع للمحتوى
