# Services

Blueprint Studio delivers structured static websites built for clarity, speed, and long-term maintainability.

Every service follows a defined workflow, controlled scope, and predictable delivery.

For detailed pricing and add-ons, see [Packages](packages.md).

## Portfolio Websites

For students and professionals who need their work to speak clearly.

Ideal when you need:

- Immediate project visibility  
- Strong first impression  
- Organized navigation hierarchy  
- Mobile-ready presentation  

These sites prioritize fast evaluation and easy long-term updates.

## Service Boundaries

Blueprint provides static website services only.

We do not provide:

- Full-stack applications  
- Database systems  
- Authentication systems  
- SaaS engineering  
- Complex backend integrations  

If your project requires dynamic systems or application logic, Blueprint is not the right fit.
