# Packages

Blueprint Studio offers three structured website packages, a CV template product, and optional add-ons.

Clear scope. Predictable delivery. Long-term maintainability.

---

## How Blueprint Defines Structure

**Section**  
A top-level navigation item in your website (e.g., Overview, Methodology, Results, Contact).

**Page**  
A single standalone content page within a section.

Example:  
If you have a “Methodology” section with 2 detailed write-ups, that equals:

- 1 section  
- 2 pages  

Sections define navigation.  
Pages define content depth.

---

### Section-to-Page Flexibility

Each tier includes a fixed number of sections.  

If your project requires fewer sections, unused sections may be converted into additional standalone pages without changing the tier price.

This allows flexible content distribution while preserving the agreed structural scope.

Conversion rate is confirmed in writing during scope review before build begins.

## Tier 1 — Personal Presence

Best for students and professionals who need a clean, organized digital presence.

**Examples:** personal digital gardens, student profiles, simple workshop landing pages.

### Includes

- Up to 4 sections (1 page per section)
- Clean responsive layout
- Structured navigation hierarchy
- Custom primary color
- Basic SEO setup (titles + metadata)
- GitHub Pages deployment
- Free GitHub domain
- 2 revision rounds
- Deployment + handover guidance

## Add-Ons

Applicable to website packages.

- Additional section — 150 SAR per section
- Bilingual setup (Arabic + English) — 300 SAR (Tiers 1 & 2)
- Custom domain setup (DNS + HTTPS) — 250 SAR
- Priority delivery — 200 SAR
- Monthly maintenance — 50 SAR / month (subscription)

### Maintenance Policy

Monthly maintenance includes ongoing content updates and minor structural adjustments within existing sections.

If a new section is required, it is billed at the standard additional section rate (150 SAR per section).

Full layout redesigns or migration to a different tier structure are quoted separately.

All add-ons are confirmed in writing before implementation.

---

## What Is Not Included

Blueprint services do not include:

- Full-stack applications
- Databases
- Authentication systems
- SaaS development
- Complex backend integrations
- Custom web applications

Projects requiring dynamic systems fall outside Blueprint’s static service scope.
