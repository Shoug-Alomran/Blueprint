# الباقات

بلو برنت ستوديو يقدم ثلاث باقات منظمة للمواقع، بالإضافة إلى منتج قوالب CV وخيارات إضافية.

نطاق واضح. تسليم متوقع. قابلية صيانة على المدى الطويل.

---

## كيف بلو برنت يعرّف الهيكلة

**القسم**  
عنصر تنقل رئيسي في الموقع (مثل: نظرة عامة، المنهجية، النتائج، تواصل).

**الصفحة**  
صفحة محتوى مستقلة داخل القسم.

مثال:  
إذا عندك قسم "المنهجية" وداخله صفحتين تفصيليتين، فهذا يعني:

- قسم واحد  
- صفحتان  

الأقسام تحدد هيكل التنقل.  
الصفحات تحدد عمق المحتوى.

---

### مرونة التحويل من قسم إلى صفحة

كل باقة تشمل عددًا ثابتًا من الأقسام.  

إذا مشروعك يحتاج أقسام أقل، تقدر تحوّل الأقسام غير المستخدمة إلى صفحات إضافية مستقلة بدون تغيير سعر الباقة.

هذا يعطيك مرونة أكبر في توزيع المحتوى مع الحفاظ على النطاق المعتمد.

نسبة التحويل يتم تأكيدها كتابيًا أثناء مراجعة النطاق قبل بدء البناء.

## الباقة 1 — حضور شخصي

مناسبة للطلاب والمهنيين اللي يحتاجون حضور رقمي مرتب وواضح.

**أمثلة:** بورتفوليو شخصي، صفحة تعريف طالب، صفحة هبوط بسيطة لورشة.

### تشمل

- حتى 4 أقسام (صفحة واحدة لكل قسم)
- تصميم متجاوب واضح
- هيكل تنقل منظم
- لون رئيسي مخصص
- إعداد SEO أساسي (عناوين + بيانات وصفية)
- نشر على GitHub Pages
- دومين GitHub مجاني
- جولتا مراجعة
- توجيه النشر والتسليم

---

## الباقة 2 — توثيق مشروع

مناسبة للمشاريع الأكاديمية اللي تحتاج تحويل إلى موقع توثيقي منظم.

**أمثلة:** مشاريع علوم الحاسب، مشاريع التخرج، تقارير هندسية منظمة.

### تشمل

- حتى 8 أقسام (تقريبًا 10–16 صفحة إجمالًا)
- تنقل مبني على فصول أو مراحل
- أقسام منظمة للنظرة العامة، المنهجية، النتائج، الملاحق، والمراجع
- دمج التقرير (اختر واحد أو الاثنين):
  - PDF مدمج
  - تقرير محوّل إلى HTML
- صفحة فريق أو مساهمين
- تصميم متجاوب واضح
- لون رئيسي مخصص
- إعداد SEO أساسي
- نشر على GitHub Pages
- دومين GitHub مجاني
- جولتا مراجعة
- توجيه النشر والتسليم

---

## الباقة 3 — توثيق بحثي

مناسبة للمشاريع البحثية الشاملة اللي تحتاج عرضًا عميقًا ومنظمًا.

**أمثلة:** مشاريع أبحاث AI/ML، دراسات متعددة المراحل، توثيق جاهز للنشر.

### تشمل

- حتى 12 قسمًا (تقريبًا 20–25 صفحة إجمالًا)
- هيكل تنقل عميق عبر مسارات متعددة
- أقسام بحثية منظمة (الأهداف، الأخلاقيات، البيانات، تفاصيل النموذج، التقييم، القيود، الأعمال المستقبلية)
- دمج التقرير/الورقة (اختر واحد أو الاثنين):
  - PDF مدمج
  - ورقة محوّلة إلى HTML
- توثيق التشغيل أو التثبيت
- توثيق البيانات أو المستودع
- صفحات المساهمين والتواصل
- تصميم متجاوب واضح
- إعداد ثنائي اللغة (عربي + إنجليزي)
- لون رئيسي مخصص
- إعداد SEO أساسي
- نشر على GitHub Pages
- دومين GitHub مجاني
- 3 جولات مراجعة
- توجيه النشر والتسليم

### القالب فقط يشمل

- ملف HTML/CSS المصدر
- توثيق التعديل
- تصميم واحد من اختيارك

لا يشمل النشر أو الدمج.

### التنفيذ الكامل يشمل

- صفحة CV مكتملة ومدمجة داخل موقعك
- نشر مباشر على مسار `/cv/`
- تنسيق متوافق مع أسلوب الموقع والتنقل

الخدمة متاحة لعملاء مواقع بلو برنت الحاليين فقط.

## الإضافات

الإضافات تنطبق على باقات المواقع.

- قسم إضافي — 150 ريال لكل قسم
- إعداد ثنائي اللغة (عربي + إنجليزي) — 300 ريال (للباقتين 1 و2)
- إعداد دومين مخصص (DNS + HTTPS) — 250 ريال
- تسليم مستعجل — 200 ريال
- صيانة شهرية — 50 ريال / شهر (اشتراك)

### سياسة الصيانة

الصيانة الشهرية تشمل تحديثات المحتوى المستمرة وتعديلات هيكلية بسيطة داخل الأقسام الحالية.

إذا احتاج المشروع قسم جديد، يتم احتسابه بسعر القسم الإضافي المعتاد (150 ريال لكل قسم).

إعادة تصميم كاملة للواجهة أو الانتقال إلى هيكل باقة مختلف تُسعر بشكل مستقل.

جميع الإضافات يتم تأكيدها كتابيًا قبل التنفيذ.

## ما لا يشمله النطاق

خدمات بلو برنت لا تشمل:

- تطبيقات Full-stack
- قواعد البيانات
- أنظمة تسجيل الدخول والمصادقة
- تطوير منصات SaaS
- تكاملات Backend معقدة
- تطبيقات ويب مخصصة

أي مشروع يتطلب أنظمة ديناميكية يكون خارج نطاق خدمات بلو برنت الثابتة.

## ملاحظات

- المدة النهائية تعتمد على جاهزية المحتوى.
- تأخر الملاحظات قد يمدد مدة التسليم.
- أي طلب خارج النطاق يتم تسعيره بشكل مستقل.
- رسوم الاستضافة وتسجيل الدومين منفصلة ما لم يُذكر خلاف ذلك.
- تحويل التقرير إلى HTML يتطلب تسليم الملف كامل قبل بدء البناء.
- تحويل الأقسام إلى صفحات يتم تأكيده خلال مراجعة النطاق.
