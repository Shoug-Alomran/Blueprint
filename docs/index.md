## Why Blueprint Works

- **Structured navigation** so people find what they need fast  
- **Static-first architecture** for speed, stability, and low maintenance  
- **Clean presentation** that feels professional (not “student template”)  
- **Easy handover** so you can update content without breaking the site  

## Scope Boundaries

| Included | Not Included |
| --- | --- |
| Static websites (MkDocs / GitHub Pages / Cloudflare Pages) | Full-stack applications |
| Information architecture + content structure | Databases |
| Responsive UI + accessibility basics | Authentication systems |
| SEO basics (titles, metadata, indexing hygiene) | SaaS product development |
| Deployment + testing + handover checklist | Custom complex animations or heavy web apps |
